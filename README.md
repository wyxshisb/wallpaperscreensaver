# Wallpaper Screensaver

让闲置的屏幕变成一幅会呼吸的画。

一个 Windows 桌面壁纸屏保应用，闲置时全屏展示你的壁纸收藏与艺术时钟。

## 功能特性

### 壁纸轮播

- 支持图片和视频混合播放（jpg/png/webp/bmp/mp4/webm/mkv/avi/mov）
- 文件夹扫描 + 单独添加图片/视频
- 4 种过渡动画：淡入、缩放、滑动、模糊
- 11 种滤镜效果：暗化、模糊、黑白、复古、鲜艳、高对比、冷色调、暖色调、反色、暗角
- Ken Burns 效果：静态壁纸缓慢平移缩放，让图片有呼吸感
- 3 种缩放模式：覆盖、包含、拉伸

### 艺术时钟

12 种预设主题，按 GPU 开销分三档：

| 分档 | 主题 | 风格 |
|------|------|------|
| 轻量 | 瑞士、粗野、模板、包豪斯 | 零 GPU 开销，纯文字/描边 |
| 均衡 | 黑色电影、活字、方尖碑、羊皮纸 | 低 GPU 开销，实底背景 |
| 华丽 | 磨砂、极光、冰川、合成波 | 高 GPU 开销，毛玻璃/液态/霓虹 |

- 拖拽调整时钟位置
- 4 种字体：系统、衬线、等宽、优雅
- 文本对齐、字号、颜色微调
- 12/24 小时制、秒数显示
- 智能配色：根据壁纸亮度自动调整文字和毛玻璃颜色

### 信息展示

- 天气信息（自动定位）
- 农历日期与节气
- 座右铭（自定义，默认 "One Step Ahead"）
- 电池状态
- 倒计时（自定义事件，到期当天脉冲发光）

### 系统集成

- 系统托盘常驻
- 闲置自动触发屏保
- 全屏应用时自动跳过
- 开机自启
- 新手教程（托盘菜单入口）

## 安装

1. 从 [Releases](../../releases) 下载最新版安装包
2. 运行 `WallpaperScreensaver-Setup-x.x.x.exe`
3. 安装完成后双击桌面快捷方式启动

## 使用

1. **添加壁纸**：双击托盘图标 → 壁纸源 → 选择文件夹或添加图片/视频
2. **选择主题**：时钟样式 → 选择预设主题 → 拖拽调整位置
3. **开启屏保**：设置闲置时间，电脑闲置后自动进入屏保
4. **退出屏保**：移动鼠标或按任意键

## 开发

### 环境要求

- Node.js >= 18
- npm >= 9

### 开发模式

```bash
# 安装依赖
npm install

# 启动开发服务器 + Electron
npm run electron:dev
```

### 构建

```bash
# 构建前端
npm run build

# 打包安装程序
npx electron-builder --win nsis --x64
```

## 技术栈

- **Electron** — 跨平台桌面应用框架
- **React 19** — UI 组件库
- **Vite** — 构建工具
- **electron-store** — 配置持久化
- **lunar-javascript** — 农历计算
